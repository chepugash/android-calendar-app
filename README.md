# Android Calendar Application
Приложение-ежедневник, позволяющее создавать, просматривать и удалять события по дате и времени

**СТЕК:**
- Single Activity
- Clean Architecture
- MVI
- Jetpack Compose
- Coroutines, Flow
- Koin
- Room
- JUnit 4, MockK
